# 更新日志

## v0.6.0

- 每张图片显示处理耗时（毫秒），状态栏显示批次总耗时
- 监听板块手动批量处理改为多线程并发
- 切换页面不再丢失文件选中状态
- 清理未实现的视频/音频/水印模块，聚焦图片处理
- 移除水印相关功能入口

## v0.5.0

- 图片缩放功能：支持最大尺寸（宽×高）和百分比两种模式
- 文件列表显示图片分辨率
- 图片预览：覆盖模式下显示单张处理后图片
- 移除历史记录页面（与现有功能重复）
- 修复缩放在按大小模式下不生效的问题
- 许可证改为 GPL-3.0

## v0.4.1

- 启动时自动检查更新，有新版本时设置图标显示红点提示
- 设置页版本行显示「有新版本可用」提示
- 修复 auto updater process 插件权限

## v0.4.0

- 应用内反馈功能：支持 Bug 报告、功能建议，截图支持粘贴/拖拽/多选（最多 3 张）
- 反馈数据提交到 Google Sheets，截图上传到 Google Drive
- 更新检查错误提示友好化
- 更新按钮 hover 样式修复

## v0.3.0

- 图片全屏预览：拖拽分割线对比原图和处理后效果
- 单文件处理进度条，显示解码、编码各阶段进度
- 压缩选项统一为「原图 / 压缩 / 按大小」三档，所有格式通用
- PNG 压缩滑块：100% = 无损（oxipng），< 100% = 有损（imagequant）
- GIF/BMP/TIFF 不支持有损/按大小时自动跳过并提示
- PNG 按大小压缩失败时返回无损结果而非报错
- 界面缩放设置（80% ~ 120%）
- 工具栏按组换行，小窗口不再重叠
- 压缩提示改为点击显示的 tooltip

## v0.2.1

- 系统通知默认关闭

## v0.2.0

- 应用名称中文化为「媒体工坊」
- 添加自动更新功能（设置页检查更新）
- PNG 压缩支持三档：原图 / 无损(oxipng) / 有损(imagequant+质量滑块)
- 默认输出目录拆分为图片、视频、音频三个独立设置
- Windows 路径兼容修复
- 文件夹监听：PNG 输出支持无损/有损选择
- 全局设置：并发处理数、文件大小单位、系统通知、开机自启

## v0.1.0

- 图片批量压缩，7 种预设（无损、轻压缩、标准、极限、目标大小、Web 优化）
- 格式互转：JPG、PNG、WebP、AVIF、GIF、BMP、TIFF
- 目标大小压缩（二分搜索最优质量）
- EXIF 剥离、文件大小过滤
- 文件夹树形视图，可折叠目录层级
- 批量选择，按文件和按文件夹勾选
- 多文件夹独立监听，每个文件夹独立启停和配置
- 文件稳定性检测，防重复处理机制
- 系统托盘实时状态显示
- 处理日志持久化
- 设置持久化
- macOS 毛玻璃 / Windows Mica 效果
- GitHub Actions CI 三平台构建
